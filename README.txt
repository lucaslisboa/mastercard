# mastercard
Web component created to enter credit card details such as card number, expiration date, security code and consumer name.

> NOTE: The project requires improvements and is not fully finalized

Project components:

* [creditcard-input](https://github.com/lucaslisboa/mastercard/blob/main/components/creditCardInput.js). A simple component that shows the rendering of a <code>card</code> and a <code>form</code> with fields to fill in like the card number, expiration date, security code and consumer name, as well as some validations.
